网站

Flush Cache

语言：英

网址：https://developers.google.com/speed/public-dns/cache

用处：刷新一个域名的 DNS 缓存，解决更改域名服务器或者 DNS 记录等很久都没有生效，Netlify 部署很有用（可以解决证书要等很久）

HTML5 UP

语言：英

网址：https://html5up.net/

用处：各种 HTML 网页模板，我发现 Linhk1606 的个人主页就是用这里面的模板

YAML Validator

语言：英

网址：https://yamlvalidator.com/

用处：可以把 YAML 复制进去检查有没有问题，之前友链页总是报 YAML 缩进错误，找到这个网站之后复制进去立马解决了

Preloaders.net

语言：英

网址：https://preloaders.net/

用处：生成加载中图片

WallpaperHub

语言：英

网址：https://wallpaperhub.app/

用处：很多好看的图片、壁纸（Microsoft 每日一图）

Image Resizer

语言：英

网址：https://imageresizer.com/

用处：调整图片的大小

CloudConvert

语言：英

网址：https://cloudconvert.com/

用处：转换各种文件，超级好用！！！

loading.io

语言：英

网址：https://loading.io/

用处：生成加载中图片

iLoveIMG

语言：支持 中 英

网址：https://www.iloveimg.com/

用处：各种图片相关工具，炒鸡好用，特别是有 GIF 工具！

PNG or JPG to SVG

语言：英

网址：https://www.pngtosvg.com/

用处：把 PNG 或 JPG 图片转换成真正的 SVG，不是 Base64！（复杂图片难免会失真）

爱站网

语言：中

网址：https://www.aizhan.com/

用处：各种关于网站、SEO 的工具，好用！

Unicode 字符百科

语言：支持 中 英

网址：https://unicode-table.com/cn/

用处：可以查到所有 Unicode 字符虽然广告有点多

web.dev

语言：英

网址：https://web.dev/

用处：借助来自 web.dev 的指导和分析，在您自己的站点和应用程序上获得 Web 的现代化功能。

Cleanmock

语言：英

网址：https://cleanmock.com/

用处：简介样机图片生成器

Bigjpg

语言：支持 中 英

网址：https://bigjpg.com/

用处：AI 人工智能无损图片放大，使用人工智能深度卷积神经网络

MDN Web 文档

语言：支持 中 英

网址：https://developer.mozilla.org/

用处：学习 Web 开发的最佳实践

Squoosh

语言：英

网址：https://squoosh.app/

用处：开源图像压缩网站

软件

WebStorm

语言：英

网址：https://www.jetbrains.com/webstorm/

用处：好用的前端 IDE（打开速度有点慢）

Mark Text

语言：英

网址：https://marktext.app/

用处：最好用的 MarkDown 编辑器

Typora

语言：支持 中 英

网址：https://typora.io/

用处：好用的 MarkDown 写作软件

Visual Studio Code

语言：支持 中 英

网址：https://code.visualstudio.com/

用处：无所不能的代码编辑器

PotPlayer

语言：支持 中 英

网址：http://potplayer.daum.net/

用处：非常强大的播放器

Imagine

语言：支持 中 英

网址：https://github.com/meowtec/Imagine

用处：用于压缩 PNG 和 JPEG 的桌面应用程序
